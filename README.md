# 📉 Estatística Aplicada — Teste Z

Análise estatística comparando a efetividade de duas estratégias de ensino de matemática por meio de um teste Z. O projeto testa hipóteses sobre diferenças de médias entre dois grupos e apresenta interpretação e visualização dos resultados.

## 🎯 Objetivo

Aplicar teste de hipótese (Teste Z) para verificar se há diferença estatisticamente significativa entre os resultados de duas estratégias de ensino de matemática.

## 🛠️ Tecnologias

`Python` `SciPy` `Pandas` `Matplotlib` `Jupyter Notebook`

## 📊 Metodologia

- Definição das hipóteses nula e alternativa
- Cálculo do teste Z para comparação de médias
- Interpretação do p-valor e nível de significância
- Visualização gráfica da distribuição e da região crítica

## 🚀 Como executar

```bash
git clone https://github.com/henry842/estatistica-aplicada-teste-z.git
cd estatistica-aplicada-teste-z
jupyter notebook
```
